# OpenTab VS Code Extension

This is a minimal VS Code extension that provides OpenTab syntax highlighting.

## Development

1. Open this folder (`editors/vscode`) in VS Code.
2. Press `F5` to launch an Extension Development Host.
3. In the new window, open or create an `.otab` file to see highlighting.

## Packaging (optional)

If you want to package the extension, install `vsce` and run:

```bash
npx vsce package
```
